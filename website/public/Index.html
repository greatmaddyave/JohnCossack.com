About Me

Who am I and why I've brought you here

Hi, I'm John. I'm a Full Stack Web Developer. What's a Full Stack Web Developer? Well, it's a programmer that can design the Front End of websites and web applications. This includes using programs and frameworks such as photoshop, HTML and CSS to gather the look and feel of your website. Some people use the term, UX & UI Design, which stands for "User Interface".

So, you're all up to speed on Front End but the other side of the "Full" title is in the...you guessed it Back End Development. So What is "Back End". Well, Back End is how the website functions. This includes, but isn't limited to, animations, log in's, log out's, server side databasing so that you can store information in your website such as a users email address as well as their name and prefrences when they log into your site as well as ecommerce...you're gonna want to accept payments right? Well Back End Development helps with all that.
